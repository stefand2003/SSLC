Accessibility Statement

This SSLC website is commited to providing a user friendly experience for all visitors, including those with disabilities. I have tried my best to ensure that all the content that is included is accessible to individuals with various special needs:

  • I have included ` alt ` text for images, making them understandable for screen readers.
  • I have used semantic HTML elements (headers, footers, main, nav, headers1,2,3.. etc) where appropriate to enhance readability to make sure that search engines interpret my website accurately.
  • I have also ensured that I use the "a tag and the input tag" for the users who are unable to use a mouse. They simply can navigate through the links and interact with the form fields using their keyboard instead.
